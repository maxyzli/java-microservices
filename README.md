# Java Microservices

This project demonstrates how to write microservices using Java and Spring Boot.

## How to Run

1. **Install Maven Build Tool**

    ```bash
    brew install maven
    ```

2. **Start the Server**

    ```bash
    mvn spring-boot:run
    ```

3. **Access the Application**

   Open your browser and navigate to:

    ```bash
    http://localhost:8080
    ```
